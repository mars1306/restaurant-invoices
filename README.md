# 🧾 Restaurant Invoices SaaS

A professional, AI-powered invoice management platform for bars and restaurants.

## ✨ Key Features
- **AI Extraction:** Uses Claude Vision to automatically extract line items from photos or PDFs.
- **SaaS Ready:** Full Multi-tenant support with secure login and private data isolation via Supabase.
- **Price Protection:** Automatically detects price hikes from suppliers and compares prices across the market.
- **Mobile Optimized:** Responsive dashboard for managers on the go.

## 🚀 Deployment
This app is designed to be deployed on **Streamlit Community Cloud**.

### 🛠 Setup
1. Clone the repo.
2. Run `./setup.sh` to install dependencies.
3. Configure your `SUPABASE_URL`, `SUPABASE_KEY`, and `ANTHROPIC_API_KEY` in the App Settings.

Developed for modern hospitality management.
